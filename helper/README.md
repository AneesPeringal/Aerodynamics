# Matplotlib draggable plot example

![Draggable plot example](./example.gif)

## Installation

```bash
$ pip install matplotlib
```

## Usage

```bash
$ python draggable_plot.py
```

- Left click to add new point
- Drag to move point
- Right click to remove point


# matplotlib でプロット上の点をドラッグする例

## 使い方

- プロット上を左クリックで点を追加
- プロット上の点をドラッグで移動
- プロット上の点を右クリックで削除
